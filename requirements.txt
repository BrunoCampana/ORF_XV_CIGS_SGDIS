Procedimentos necessários para funcionamento
O seguinte passo-a-passo é feito para o ubuntu 16.04

1)Instalar servidor apache2:
	sudo apt install apache2

2)Instalar PHP 7.0:
	>sudo apt-get install python-software-properties
	>sudo add-apt-repository ppa:ondrej/php-7.0
	>sudo apt-get update
	>sudo apt-get purge php5-fpm
	>sudo apt-get install php7.0-cli php7.0-common libapache2-mod-php7.0 php7.0 php7.0-mysql php7.0-fpm php7.0-curl php7.0-gd php7.0-bz2

3)Clonar projeto do repositório SisCAc:
	>sudo git clone https://gitlab.com/F3l1p3/SisCAc.git

4)Instalar o MySQL Server:
	>sudo apt install mysql-server

5)Criar banco com o nome 'ccfexBeta':
	mysql>create database ccfexBeta;

5)Importar os dados:
	>mysql -uroot -p ccfexBeta < dump.sql
